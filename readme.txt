Análise de Séries Temporais e Indicadores Técnicos: Caso Magalu (MGLU3)

Este projeto utiliza Python para analisar o comportamento das ações da Magazine Luiza (MGLU3) ao longo do ano de 2021. O foco principal foi a aplicação de indicadores de tendência e a visualização avançada de dados financeiros para identificar padrões de mercado.

Principais Implementações:

    Análise de Fechamento: Monitoramento da variação do valor da ação e identificação de tendências de queda/alta no período.

    Médias Móveis (Moving Averages): Implementação de médias móveis (curto e longo prazo) para suavizar a volatilidade e identificar pontos de cruzamento e suporte/resistência.

    Gráficos de Candlestick: Construção de gráficos de velas (Candlesticks) para análise detalhada da abertura, fechamento, máximas e mínimas diárias.

    Visualização com Plotly/Matplotlib: Domínio de bibliotecas para gerar gráficos interativos e estáticos de alta precisão técnica.
